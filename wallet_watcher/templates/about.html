The Wallet Watcher Project

    The purpose of this project is to demonstrate my abilities to be a Full-Stack Web Engineer. At the front-end, I built the website with Bootstrap. The database I chose mongoDB because of its flexible schema and its high scalability. At the back-end, the whole application was written in Python and used Flask as the framework. Last but not least, I utilized Gunicorn and Nginx to deploy the project on a server in the Google Cloud Platform.
